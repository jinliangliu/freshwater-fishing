# 贡献指南 (Contributing Guide)

感谢你对淡水野钓顾问技能的兴趣！我们欢迎各种形式的贡献。

## 🎯 欢迎贡献的方向

### 1. 数据完善
- **新增鱼种**：补充更多中国淡水鱼种（需提供科学依据）
- **优化评分算法**：改进适宜度计算模型
- **补充钓位推荐**：增加更多水域的钓位建议
- **完善月相规则**：细化不同鱼种的月相偏好

### 2. 文档改进
- 修正错误或歧义
- 优化示例和说明
- 补充使用场景
- 翻译为其他语言

### 3. 功能增强
- 新增作钓方式（如路亚）
- 扩展水域类型
- 增加更多安全提示
- 优化输出格式

## 📝 提交规范

### Pull Request 要求

1. **分支命名**
   ```
   feature/新增功能描述
   fix/修复问题描述
   docs/文档更新
   ```

2. **提交信息格式**
   ```
   [类型] 简短描述

   详细描述（可选）
   - 点 1
   - 点 2
   ```

   类型：`feat`、`fix`、`docs`、`style`、`refactor`、`test`

3. **代码/文档规范**
   - 遵循现有文档格式
   - 使用中文（除非是代码变量名）
   - 提供数据来源或经验依据
   - 更新版本历史

### 数据贡献要求

- **鱼种数据**：需提供科学名称、分布区域、生态特征
- **评分规则**：需提供实验数据或长期经验总结
- **安全规则**：必须基于权威来源或实际案例

## 🔍 代码审查

所有 PR 将经过以下审查：

1. **准确性**：数据是否准确可靠
2. **一致性**：是否符合现有文档风格
3. **安全性**：是否引入安全隐患
4. **实用性**：是否真正提升用户体验

## 🚫 不接受的贡献

- 商业推广内容
- 未经证实的钓鱼技巧
- 违反环保理念的建议
- 可能危害生态平衡的内容
