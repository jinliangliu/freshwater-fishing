## 📖 简介

这是一个基于 OpenClaw 的淡水钓鱼顾问技能，专为中文钓鱼爱好者设计。根据天气、季节、月相、水域类型、目标鱼种，提供专业、科学的作钓建议。

### ✨ 核心特性

- **3 种作钓方式**：台钓、传统钓、筏钓
- **30+ 鱼种支持**：覆盖中国全域淡水本土鱼种和常见入侵鱼种
- **9 种水域类型**：江河、湖泊、城市河道、黑坑、水库、闸口、山涧、溪流、鱼塘
- **智能季节推断**：基于农历月份自动判断季节（符合中国传统）
- **月相影响评估**：新月/满月等月相对鱼情的影响规则
- **天气 - 鱼情对应**：6 种典型天气下的鱼情分析和应对策略
- **安全否决机制**：8 项安全否决 + 4 项强烈不建议
- **环保理念**：文明垂钓、垃圾清理、保护水域、放流原则

## 🚀 快速开始

### 前提条件

- 已安装 [OpenClaw](https://github.com/openclaw/openclaw)
- 配置了 `weather` 技能（用于获取天气数据）

### 安装方法

1. 克隆本仓库或下载 `SKILL.md` 文件
2. 将 `freshwater-fishing` 目录放入你的 OpenClaw 技能目录：
   ```bash
   cp -r freshwater-fishing ~/.openclaw/workspace/skills/
   ```
3. 重启 OpenClaw 或运行 `/reload` 命令

### 使用示例

**基础用法**：
```
明天杭州千岛湖适合台钓鲫鱼吗？
```

**指定鱼种**：
```
周末上海青浦城市河道筏钓翘嘴怎么样？
```

**多参数组合**：
```
周六凌晨 5 点去千岛湖水库夜钓黄颡鱼，适合吗？
```

**自定义季节**：
```
明天去太湖钓草鱼，夏季，台钓，白天
```

## 🎯 支持的鱼种

### 本土鱼种（优先保护）
鲫鱼、鲤鱼、草鱼、鲢鳙、翘嘴、鳊鱼、黄颡、鲶鱼、马口、鳟鱼、鳜鱼、黑鱼、青鱼、中华倒刺鲃、赤眼鳟、黄尾鲴、餐条、麦穗鱼、棒花鱼、光唇鱼、宽鳍鱲、团头鲂、鲮鱼等

### 入侵鱼种（严禁放生）
罗非、大口黑鲈、太阳鱼、淡水白鲳、鳄雀鳝等

### 濒危鱼种（发现即放流）
唐鱼、白云山鱼

> ℹ️ **重要提示**：入侵鱼种严禁放生到自然水域！濒危鱼种发现即放流！

## 🌊 支持的水域类型

| 水域 | 特点 | 推荐钓位 |
|------|------|---------|
| 江河 | 流水溶氧足 | 回水湾、洄流区、桥墩旁 |
| 湖泊 | 静水/缓流 | 水草边、铧尖、深浅交界 |
| 城市河道 | 水流稳定 | 路桥下、闸口旁、水草区 |
| 黑坑 | 人工养殖 | 塘角、增氧机旁、投饵区 |
| 水库 | 水深面积大 | 大坝附近、湾汊、老钓点 |
| 闸口 | 水流交换好 | 闸口上下游 5-20 米 |
| 山涧 | 冷水高溶氧 | 缓流区、石缝旁 |
| 溪流 | 流水鱼活性高 | 洄流区、石后、入水口 |
| 鱼塘 | 人工养殖 | 投饵区、增氧机旁、塘角、坝口 |

**通用钓位**：树荫下（夏季）、桥墩旁（掠食性鱼）、坝口（水库）

## 📅 季节划分（农历标准）

- **春季**：农历正月、二月、三月
- **夏季**：农历四月、五月、六月
- **秋季**：农历七月、八月、九月
- **冬季**：农历十月、十一月、十二月

> 若未提供 `season` 参数，系统会自动根据日期推算农历并判断季节。

## 🌙 月相影响

| 月相 | 农历日期 | 夜钓修正 | 昼钓修正 |
|------|---------|---------|---------|
| 新月 | 初一 | +10 分 | 无影响 |
| 上弦月 | 初七、初八 | 前夜 -5 分/后夜 +5 分 | 无影响 |
| 满月 | 十五、十六 | -10 分 | +5 分 |
| 下弦月 | 廿二、廿三 | 前夜 +5 分/后夜 -5 分 | 无影响 |
| 残月 | 月末/月初 | +5 分 | 无影响 |

**特殊规则**：
- 初一、十五「大鱼日」：全天 +5 分
- 月出月落窗口：掠食性鱼 +10 分

## 🌤️ 天气 - 鱼情对应

| 天气 | 鲫鱼活性 | 应对策略 |
|------|---------|---------|
| 晴天无风 | 差 | 钓深水，长竿，中午收竿 |
| 多云/阴天 | 良 | 全天可钓，3 米内浅水 |
| 小雨天 | ⭐极佳 | 全天狂口，进水口、水草边 |
| 大雨后 | 好 | 钓淹没区、浑水边缘 |
| 闷热低压 | 极差 | 收竿回家 |
| 北风降温 | 好 | 全天可钓，选下风口 |

## ⚠️ 安全否决项

触发以下任一条件，适宜度直接归零：

1. 雷暴、强对流天气
2. 风力 ≥ 6 级
3. 夜间温度 < 5℃ 且无御寒装备
4. 冰雹、暴雪等极端天气
5. 洪水期/水质异常
6. **高压电线附近**（致命风险）
7. **野外用火**（防火期严禁）
8. **水边无安全防护**（防溺水）

## 🌱 环保倡议

- **文明出钓**：主动清理垃圾，不乱扔
- **带走垃圾**：离开时带走所有垃圾
- **保护水域**：不破坏水草、不使用有害饵料
- **文明垂钓**：不影响他人，不占多钓位
